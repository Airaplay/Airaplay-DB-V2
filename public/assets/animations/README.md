# Animation Assets

## Done.gif

Place your Done.gif file in this directory.

**File Requirements:**
- File name: `Done.gif`
- Recommended size: 120x120px to 160x160px
- File size: Under 500KB for optimal mobile performance
- Format: GIF with animation

**Usage:**
This animation is displayed when users successfully withdraw treats to their live wallet in the Treat System.

**Path Reference:**
The component references this file at: `/assets/animations/Done.gif`

**Fallback:**
If the GIF fails to load, the system will automatically fallback to a green checkmark icon.
